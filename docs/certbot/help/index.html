    得到帮助¶

    在这一页我能找到什么?¶

    本页包含学习如何打开 https 以及 Certbot 在此过程中扮演的角色的资源。 我们认为它们非常有用!

    如果他们没有回答你的问题，在 https://community.letsencrypt.org/ 上有一个熟悉项目的大社区，他们可能有你正在寻找的答案。

    了解 Certbot 和 HTTPS¶

    正在寻找配置 Certbot 的方法?阅读 Certbot 文档。

    Certbot开发人员和志愿者在阅读文档中为本文档做出了贡献。

    不知道如何设置你的网络服务器?查看外部教程。

    DigitalOcean提供了在特定操作系统上设置web服务器的很好的指南，无论您的托管提供商是什么。 尝试搜索你的操作系统与“web服务器”，“nginx”，或“apache”在他们的教程页面.

    想知道一个特定的术语是什么意思?查看我们的词汇表。

    我们创建了一个术语表，这些术语对理解HTTPS生态系统很重要。

    正在寻找一个提供免费 HTTPS 证书而不需要安装 Certbot 的共享托管提供商?

    请访问我们的志愿者支持的HTTPS共享托管提供商页面。 如果您没有看到您的托管提供商或注意到过时的信息，请通过打开拉请求来更改我们网站Github repo 上的此json文件与我们联系。 您还可以找到成功编辑托管提供商列表的技巧。

    安装了 Certbot，但不管用?试试这些外部资源。

    当试图请求证书时，从Let's Encrypt得到验证错误?试试Let's Debug或check-your-website。

    证书已经安装，但网站没有按预期工作?试试whynopadlock, check-your-website，或者SSL Labs。

    关于这些站点是什么以及它们的最佳用例的更多信息可以在我们的FAQ中找到。

    对你的设置有一个具体的问题，比如……

    从专用主机提供商购买了一个VPS，不确定什么适合你?

    寻找如何让你的网站运行起来?

    寻找可能与您的用例相关的外部教程?

    看看我们的常见问题中是否已经涵盖了它

    找不到你需要的东西?¶

    把你的问题写在 Let 's Encrypt 社区论坛上。

    Let 's Encrypt和Certbot社区或许能够回答您的问题。

    查看 Certbot 源代码。

    Certbot源代码可在GitHub上获得。在Github上提交拉请求之前，请在Mattermost上与我们交谈。
